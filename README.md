# mediata-ezpage-fieldtype-bundle

## Installation

Add repository to `composer.json`. 
```js
  "repositories":[
      {
          "type": "vcs",
          "url": "git@github.com:mediatacom/mediata-ezpage-fieldtype-bundle.git"
      }
  ]
```
Require the bundle.
```bash
composer require mediatacom/mediata-ezpage-fieldtype-bundle
```
